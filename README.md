# PowerBI-Sales-Dashboard
Power BI dashboard with 8 pages including sales, customer insights, KPIs, and trends

## Overview
This project is an interactive Power BI dashboard with 8 report pages.

## Features
- Sales Performance Analysis
- Customer Segmentation
- Regional Insights
- KPI Tracking
- Trend Analysis

## Tools Used
- Power BI Desktop
- DAX
- SQL
